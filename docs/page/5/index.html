White House releases first ever open source budget proposal

03.02.2015 00:00
The White House just released the first ever open source budget proposal. It is released on GitHub, and it’s a bunch of CSV files. This is not very difficult, it requires only a few extra clicks when exporting an Excel spreadsheet, but hosting it on GitHub also opens it up to Pull Requests, which I’ve talked about before as being a much better tool for 21st century democracy. Instead of paper and a bunch of politicians in a room following procedure, we should intead have a digital system where all citizens can contribute as easily as they can update a facebook status or apply an instagram filter. Parsing nested expressions using Bison

27.01.2015 00:00
I modified my tipcalc program to handle expressions of arbitrary depth, so now it can handle input like ((($100 + 2%) + 2%) - 3%) + 3.5%. The trick was to change the start symbol to match binary_expression, and then define binary_expression recursively, like so: binary_expression: dollars OP_PLUS percentage | dollars OP_MINUS percentage | LPAREN binary_expression RPAREN OP_PLUS percentage | LPAREN binary_expression RPAREN OP_MINUS percentage This is what makes this new version a context-free grammar and not a regular grammar.
