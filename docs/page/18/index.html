Fixed point in ruby hash function

A fixed point of a function \( f:S \to S \) is an element \(x \in S\) such that {% latex %} $ f(x) = x $ {% endlatex %} That is, \(f\) is a no-op on \(x\). Some examples: The identity function on any set has all points as fixed points The absolute value function has any positive real number as a fixed point The base64 encoding function has a string as a fixed point (Check out that link above, fmota wrote about how they discovered a fixed point in the base64 encoding function, it’s very interesting) Read more...
