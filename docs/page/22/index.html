Fixing spaces in filenames

19.11.2012 00:00
Sorry this has taken so long, I’ve been working on the Cryptography Coursera class, to get rid of all spaces in all files below the current directory, the following bash script will do: for filename in $(find .); do newfilename=$(echo $filename | sed 's/ /_/g'); mv $filename $newfilename done
